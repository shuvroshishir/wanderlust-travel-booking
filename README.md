## Wanderlust Travel Booking
This is a travel booking application built with Next.js and Tailwind CSS. It allows users to browse travel destinations, make bookings, and manage their reservations. The application also includes an admin panel for managing destinations and bookings.
### Features
- Browse travel destinations with detailed information and images.
- Make bookings for selected destinations.
- View and manage user bookings.
- Admin panel for managing destinations and bookings.
### Technologies Used
- Next.js: A React framework for building server-side rendered applications.
- Tailwind CSS: A utility-first CSS framework for styling the application.
- React: A JavaScript library for building user interfaces.
